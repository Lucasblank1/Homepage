# Portfólio Pessoal - Lucas Blank

## Sobre o Projeto

Portfólio profissional desenvolvido para apresentar projetos, habilidades e experiências como desenvolvedor web. O site apresenta design moderno e responsivo, otimizado para proporcionar a melhor experiência ao usuário em diferentes dispositivos.

## Características Principais

- **Design Moderno** - Interface com glassmorphism, gradientes e animações suaves
- **Responsividade Completa** - Adaptado para desktop, tablet e dispositivos móveis
- **Performance Otimizada** - Carregamento rápido e transições fluidas
- **Efeitos Visuais** - Hover effects, sombras coloridas e animações CSS
- **SEO Otimizado** - Meta tags completas para melhor ranqueamento nos buscadores
- **Acessibilidade** - Seguindo boas práticas de acessibilidade web

## Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e otimizada
- **CSS3** - Estilização avançada com variáveis CSS, flexbox e grid
- **JavaScript** - Interatividade e navegação suave
- **Google Fonts** - Tipografia Poppins
- **Bootstrap Icons** - Biblioteca de ícones vetoriais

## Sistema de Design

### Paleta de Cores
- **Primary:** `#007bff` - Azul principal
- **Primary Dark:** `#0056b3` - Azul escuro
- **Accent:** `#00d4ff` - Ciano (destaque)
- **Background:** Gradiente de preto com imagem de fundo

### Tipografia
- **Fonte:** Poppins (Google Fonts)
- **Tamanhos:** 16px (base) até 42px (títulos)
- **Pesos:** 300 (light) até 700 (bold)

## Instalação e Uso

1. Clone o repositório:
   ```bash
   git clone https://github.com/Lucasblank1/Homepage.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Homepage
   ```

3. Abra no navegador:
   - Clique duas vezes no arquivo `index.html`
   - Ou utilize um servidor local como Live Server ou http-server

## Seções do Site

### Página Principal (index.html)
- **Hero Section** - Apresentação inicial com foto e call-to-action
- **Sobre** - Biografia, experiências e stack de tecnologias
- **Contato** - Formulário para contato direto
- **Footer** - Links para redes sociais e informações de contato

### Página de Projetos (projetos.html)
- Grid responsivo com cards de projetos
- Sistema de badges indicando status do projeto
- Links diretos para demonstração e repositório GitHub
- Categorização por tecnologias utilizadas

## Funcionalidades Implementadas

- Menu de navegação fixo com efeito glassmorphism
- Dropdown animado para acesso às redes sociais
- Botões com gradiente e efeito de brilho ao passar o mouse
- Imagens de perfil com bordas iluminadas e animação de flutuação
- Formulário de contato com validação visual
- Navegação suave entre seções da página
- Transições e animações em elementos interativos
- Sistema de ícones para tecnologias com efeitos hover

## Breakpoints de Responsividade

- **Desktop:** Acima de 1024px - Layout completo com todos os recursos
- **Tablet:** 768px a 1024px - Layout adaptado com ajustes de espaçamento
- **Mobile:** Até 768px - Layout em coluna única otimizado para toque
- **Small Mobile:** Até 480px - Otimizações adicionais para telas pequenas
